# 科技企业官网

基于 Vue 3 + Vite + Tailwind CSS + DaisyUI 构建的现代化科技企业官网。

## 技术栈

- [Vue 3](https://v3.vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [DaisyUI](https://daisyui.com/)

## 功能特性

- 响应式设计，适配各种设备屏幕
- 现代化 UI 界面，使用 DaisyUI 组件库
- 深色/浅色主题切换
- 首页、产品、解决方案、关于我们、联系我们等页面
- 优化的用户体验和交互设计

## 安装与运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 构建生产版本：
   ```bash
   npm run build
   ```

4. 预览生产版本：
   ```bash
   npm run preview
   ```
